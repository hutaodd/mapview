### 环境说明

在安装好python后，在当前文件夹，shift+右键，在当前目录调出命令提示符输入一下命令安装依赖

`pip install -r requirements.txt `

### 运行说明

在安装好运行环境后，继续在当前命令提示符目录下运行以下命令

`streamlit run map.py`

命令运行后将自动打开浏览器，按提示使用即可
